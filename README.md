# Homebrew Tap for PWA-Kit

This is the official Homebrew tap for [PWA-Kit](https://github.com/chindris-mihai-alexandru/PWA-Kit) - The macOS toolkit for progressive web apps.

## Installation

```bash
# Add the tap
brew tap chindris-mihai-alexandru/pwakit

# Install PWA-Kit
brew install pwa-kit
```

## Quick Start

```bash
# Create a PWA from any website
pwakit create https://gmail.com --name Gmail

# List all PWAs
pwakit list

# Remove a PWA
pwakit remove Gmail
```

## About PWA-Kit

PWA-Kit creates lightweight, WebKit-powered progressive web apps for macOS with:

- **<100MB RAM** per app (vs Safari's 200-300MB)
- **Native integration** with Dock, Launchpad, Spotlight
- **Web standards** support (Web Push, Badging, Service Workers)
- **Open source** (GPL-3.0)

Learn more at: https://github.com/chindris-mihai-alexandru/PWA-Kit
